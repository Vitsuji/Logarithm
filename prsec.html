Logarithm is secure

Logarithm is serious when dealing with privacy and security. Logarithm keeps all of the users information fully secured and will deal with any threats immediately.

Privacy

Logarithm will not reveal any personal data about the user under any circumstances. The inputed data of the user by signing up is secure and only the username will be revealed to the public. Private chats that the user makes will not be checked by staff, meaning privacy in such chats. Logarithm will ask no more personal information than the values inputed by the user while registering.

Security

Everything is secure while using Logarithm. Logarithm is very serious about user privacy and security. All of the user information is stored with maximum security and no 3rd party could obtain personal information and harm the user. Logarithm has people who check if there are no signs of hacking, inappropriate content and discrimination. Furthermore, the user can report anything they deem prejudice, unwanted or a hacking attempt such as phishing, this would help the experience of other users. Any illegal activities occuring on Logarithm will undoubtely be reported to the authorities.
